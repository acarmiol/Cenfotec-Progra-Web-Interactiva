Tarea 6 Custom polymer

Hacer un custom polymer que funcione como un dashboard de ventanas. El polymer contenedor debe de usar bootstrap, precisamente la característica del grid, para funcionar responsive. El polymer tipo ventana debe poder aceptar cualquier html de contenido y usar jqueryui, específicamente la propiedad dragable, para poder moverse e intercambiarse
